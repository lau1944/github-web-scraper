# github-web-scraper
A Github Web Scraper Rest Api Using Flask

## Installation
```
pip install flask
pip install beautifulsoup4
```

## Run
open console, type **python scraper.py**

Open the default URL on browser 


## Path 

### Trending
 ** - [/github/trend] : Retrieve trending open source projects from github **
